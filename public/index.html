API Endpoints

POST /api/agent/:userId/chat

Send a message to the AI coding agent

curl -X POST https://dev.vibez.now/api/agent/USER_ID/chat \
  -H "Content-Type: application/json" \
  -d '{
    "message": "Create a hello.js file that prints Hello World"
  }'

Response:

{
  "success": true,
  "reply": "I'll create a hello.js file for you...",
  "usage": {
    "inputTokens": 1234,
    "outputTokens": 567,
    "totalTokens": 1801
  },
  "toolCalls": 1,
  "conversationLength": 2
}

GET /api/agent/:userId/history

Retrieve conversation history for a user

curl https://dev.vibez.now/api/agent/USER_ID/history

DELETE /api/agent/:userId/history

Clear conversation history for a user

curl -X DELETE https://dev.vibez.now/api/agent/USER_ID/history

GET /api/agent/status

Check if the agent is available

curl https://dev.vibez.now/api/agent/status

JavaScript Example

async function askAgent(userId, message) {
  const response = await fetch('https://dev.vibez.now/api/agent/' + userId + '/chat', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ message })
  });

  const data = await response.json();
  return data.reply;
}

// Usage
const reply = await askAgent('user123', 'Create a calculator.js file');
console.log(reply);
