Authentication

All agent endpoints require authentication. First, register or login to get your access token.

POST /api/auth/register

Register a new user account

curl -X POST https://dev.vibez.now/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "you@example.com",
    "password": "yourpassword",
    "username": "YourName",
    "vibezUserId": "optional-vibez-user-id-from-frontend"
  }'

Note: The vibezUserId parameter is optional. If provided, it will link this account to your Vibez.now mobile app user. If omitted, a new user ID will be generated.

Response:

{
  "success": true,
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",

    "email": "you@example.com",

  }
}

Agent API Endpoints

Note: All agent endpoints require the Authorization: Bearer YOUR_TOKEN header.

POST /api/agent/chat

Send a message to the AI coding agent (uses your authenticated user ID)

curl -X POST https://dev.vibez.now/api/agent/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{
    "message": "Create a hello.js file that prints Hello World"
  }'

Response:

{
  "success": true,
  "reply": "I'll create a hello.js file for you...",
  "usage": {
    "inputTokens": 1234,
    "outputTokens": 567,
    "totalTokens": 1801
  },
  "toolCalls": 1,
  "conversationLength": 2
}

GET /api/agent/history

Retrieve your conversation history

curl https://dev.vibez.now/api/agent/history \
  -H "Authorization: Bearer YOUR_TOKEN"

DELETE /api/agent/history

Clear your conversation history

curl -X DELETE https://dev.vibez.now/api/agent/history \
  -H "Authorization: Bearer YOUR_TOKEN"

GET /api/agent/status

Check if the agent is available

curl https://dev.vibez.now/api/agent/status

JavaScript Example

// First, login to get your token
async function login(email, password) {
  const response = await fetch('https://dev.vibez.now/api/auth/login', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ email, password })
  });
  const data = await response.json();
  return data.token;
}

// Then use the token to interact with the agent
async function askAgent(token, message) {
  const response = await fetch('https://dev.vibez.now/api/agent/chat', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${token}`
    },
    body: JSON.stringify({ message })
  });

  const data = await response.json();
  return data.reply;
}

// Usage
const token = await login('you@example.com', 'yourpassword');
const reply = await askAgent(token, 'Create a calculator.js file');
console.log(reply);
